# 🚗 End-to-End SQL Data Analysis Project

## 📌 Project Overview

This project analyzes an Automobile Dataset using MySQL to answer real-world business questions.

## 📊 Dataset

- Source: Kaggle
- Automobile Dataset

## 🛠 Tools Used

- MySQL
- SQL
- Kaggle Dataset

## 📈 SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- LIMIT
- DISTINCT
- BETWEEN
- AVG()
- COUNT()

## 📋 Business Questions

- Top 5 fuel-efficient cars
- Average horsepower by origin
- Average MPG by cylinders
- Heaviest car
- Cars manufactured in 1975
- Cars with horsepower between 100–150
- And more...
