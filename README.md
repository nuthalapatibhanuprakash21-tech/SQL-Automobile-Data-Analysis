## 📌 Project Overview

This project demonstrates SQL-based data analysis on a real-world Automobile Dataset from Kaggle.

The objective is to answer business questions using SQL and gain insights into vehicle performance, fuel efficiency, engine specifications, and manufacturing trends.

---

## 📂 Dataset

**Source:** Kaggle Automobile Dataset

Dataset includes:

- Car Name
- MPG
- Cylinders
- Displacement
- Horsepower
- Weight
- Acceleration
- Model Year
- Origin

## 📈 Business Questions Solved

1. Cars with MPG above 30
2. Average horsepower by origin
3. Count cars by origin
4. Top 5 fuel-efficient cars
5. Distinct cylinder counts
6. Average weight by cylinders
7. Cars accelerating in under 10 seconds
8. Average MPG by model year
9. Heaviest car
10. Cars with horsepower between 100–150
11. Average MPG by cylinders
12. Cars manufactured in 1975

---

## 📊 Key Insights

- Japanese cars achieved higher fuel efficiency.
- 4-cylinder vehicles delivered better mileage.
- Vehicle weight generally increased with cylinder count.
- Fuel efficiency improved in later model years.
- Horsepower varied significantly across regions.
